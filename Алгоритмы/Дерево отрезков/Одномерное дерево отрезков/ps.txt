Вводится число N (размер массива) и N чисел (элементы массива).
Далее вводится число Q - число запросов - и Q запросов одного из двух видов:
1) "sum L R", где 0 <= L <= R <= N - 1.
   Программа должна вывести сумму чисел на отрезке [L; R].
2) "add I D", где 0 <= I <= N - 1, D - целое число.
   Программа должна увеличить I-тый элемент массива на величину D (которая может быть и отрицательной).
