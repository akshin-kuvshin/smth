Абай принес вам простую задачу без легенды.

Задан массив AA, состоящий из NN натуральных чисел, а также QQ запросов вида L,RL,R. Ответом на запрос является максимальное целое KK (K≥0K≥0), что для него найдется натуральное XX, при котором числа XX, 2X2X, 4X4X, ..., 2K⋅X2K⋅X встречаются среди чисел ALAL, AL+1AL+1, ..., ARAR. Ваша задача – посчитать ответ на каждый запрос.

Примечание: натуральным называется целое число больше нуля.

Входные данные

В первой строке выходных данных заданы два натуральных числа NN и QQ (1≤N,Q≤5⋅1051≤N,Q≤5⋅105).

Во второй строке задается массив AA (1≤Ai≤10181≤Ai≤1018).

Начиная с третьей строки, задаются запросы LL, RR (1≤L≤R≤N1≤L≤R≤N).

Каждый запрос задан в отдельной строке.

Выходные данные

Выведите QQ строк, в ii-й строке должен быть ответ на ii-й запрос.
