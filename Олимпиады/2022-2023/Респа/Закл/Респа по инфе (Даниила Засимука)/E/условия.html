Дан массив из nn чисел, а также mm запросов. Каждый запрос содержит два числа ll и rr (1≤l≤r≤n1≤l≤r≤n). Необходимо для каждого запроса вычислить сумму всех подмассивов f(a[x...y])Tf(a[x...y])T от ll до rr, где a[x...y]a[x...y] - это часть исходного массива, начинающаяся с xx и заканчивающаяся yy (l≤x≤y≤rl≤x≤y≤r), то есть массив [ax,ax+1,...,ayax,ax+1,...,ay].

Для массива bb длины kk, функция f(b)f(b) находит массив cc длины kk, который представляет собой префикс-максимумы массива bb, а затем считает количество уникальных чисел в массиве cc.

Более формально, пусть cici = max(b1,b2,...,bib1,b2,...,bi). Тогда f(b)f(b) равна количеству уникальных чисел в массиве cc.

Например, для массива b=[3,1,4,1,5,9,2,6,5]b=[3,1,4,1,5,9,2,6,5], мы получаем массив префикс-максимумов c=[3,3,4,4,5,9,9,9,9]c=[3,3,4,4,5,9,9,9,9]. Затем мы считаем количество уникальных чисел в cc, которое равно 44 (3,4,53,4,5 и 99).

Ваша задача - написать программу, которая будет для каждого запроса будет находить сумму всех его подмассивов.

Так как ответ может быть очень большим, выведите его по модулю 109+7109+7.
